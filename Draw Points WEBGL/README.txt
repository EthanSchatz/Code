Go into the Draw points folder
and run the HTML file,
It will open a grey canvas where you
can draw points by first holding down
the letter of the color oyu want to draw
and then left clicking.

COLORS
     {
	b:blue
	r:red
	w:white
	c:cyan
	m:magenta
	g:green
	y:yellow
     }
To delete a dot, simply right click it.

To select a dot (bring it to the front),
simply left click it.

To move a dot, left click and drag it.

For added fun, hold down the 1 key to repeatedly shift the color of all dots.
